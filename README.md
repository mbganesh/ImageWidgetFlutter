### ASSET
```

 Image.asset(
            image1,         //  asset path
            height: 400,
            width: 250,
            fit: BoxFit.cover,
          ),
          
```

### NETWORK
```

   Image.network(
            src,         //  image url
            height: 150,
            width: 150,
          ),
          
```
